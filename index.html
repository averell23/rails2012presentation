Rails Cheats

  • rails new <app_name> Creates a new Rails app
  • bundle install Installs dependencies
  • rails server Starts the development server
  • rails console Fires up the rails console

Model Cheats

  • rails generate model <foo> Generates Model and Migration
  • rake db:migrate Runs the pending db migrations
  • rake db:migrate:redo Reruns the last migration
  • Foo.find(id) Finds the record with that id
  • Foo.new(params) Creates a new record
  • foo.save Saves a record
  • foo.update_attributes(params) Updates a record

Controller Cheats

  • rails generate controller <foos> Generates the controller
  • @controller_var = some Variable for controller and views
  • render :text => "Some" Renders text directly to the page
  • redirect_to some_url Redirects to another page

View Cheats

  • <% ruby_code %> Ruby code in a template
  • <%= ruby_code %> Puts the result of ruby_code into the page
  • form_for(@foo) Creates a form for a model object
